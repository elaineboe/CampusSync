
CampusSync Design System and UI Schema

1. Design Philosophy

The interface must reflect:
Low cognitive load
Clear academic structure
Professional university system appearance
Simple navigation
Minimal configuration

The interface must prioritise clarity and information visibility.


2. Visual Identity

Logo
CampusSync logo represents academic scheduling and coordination.
Visual concept: Calendar icon combined with connected nodes representing synchronisation between students and lecturers.
